# 🖥️ Required! Install git, node and yarn (or npm)
```
# Clone Repository
$ git clone https://github.com/GuilhermeProenca/Next-Level-Week-5.git

# Go to server folder
$ cd Next-Level-Week-5

# Install Dependencies
$ yarn

# Run Aplication (Terminal 1)
$ yarn dev

# Run Server (Terminal 2)
$ yarn server

# Access localhost
http://localhost:3000
```
